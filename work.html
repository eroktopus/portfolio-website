React meetApp is a serverless, progressive web application built with React, utilizing the Google Calendar API for fetching and sorting upcoming events in different cities.

  • Enables users to effortlessly search and filter events by city, providing real-time access to upcoming events and detailed event information directly through the Google Calendar service.
  • Utilizes service workers to allow offline functionality, ensuring that users can access previously loaded event data and interact with the app even in the absence of an internet connection.
  • Designed as a Progressive Web App (PWA) to ensure a seamless and responsive user experience across all devices, with features like "add to home screen" for easy access and push notifications for updates.
  • Incorporates interactive data visualization tools to display event details graphically, with charts showing the distribution of events by city and the popularity of event genres, enhancing user engagement and data comprehension.
